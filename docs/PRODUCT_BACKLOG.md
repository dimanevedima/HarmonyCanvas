# Harmony Canvas — продуктовый backlog после отделения

Источник: проверка первого работающего VST3 в Ableton Live 12, 22 июля 2026.

## P0 — DAW и базовый editor workflow

- синхронизировать tempo с Ableton Live; ручной BPM становится fallback;
- минимизировать верхнюю панель до Play/Stop, tonic, mode и meter;
- добавить Undo и Redo с клавиатурными shortcuts;
- отделить Loop On/Off от выбора границ;
- растягивать loop region за левый и правый край;
- добавить вертикальный scroll piano roll по всему MIDI-диапазону;
- удалить нижние analysis/reference/MIDI-секции из plugin mode;
- сделать компактный текстовый ввод аккордов.

## P1 — гармония и многоголосие

- улучшить harmony palette и chord lane;
- поддержать drag аккорда из palette прямо в chord lane;
- поддержать до четырёх melody voices;
- активный голос редактируется и имеет полный контраст;
- остальные голоса видны приглушённо и защищены от случайного редактирования;
- добавить явный voice selector и keyboard shortcuts 1–4.

## P2 — экспорт и drag-and-drop

- отдельный drag-out только аккордов;
- отдельный drag-out активного melody voice;
- drag-out всех melody voices;
- визуально различать drag handle и обычную кнопку;
- всегда оставлять альтернативную команду Export для пользователей без drag.

## UX-ограничения

- loop handles и chord/note resize не требуют пиксельной точности;
- drag начинается только после небольшого порога движения;
- критические операции имеют видимую кнопку и keyboard alternative;
- piano roll получает один предсказуемый вертикальный scroll region;
- playhead, loop и selection не должны конкурировать за один жест;
- интерактивные элементы имеют заметные focus/pressed/disabled states;
- цвет роли дополняется degree/role label и не является единственным сигналом.

## Не входит в этап отделения

Этап отделения считается чистой миграцией: существующие edit-команды и
визуальный результат должны остаться эквивалентными. Пункты выше реализуются
после автономного запуска без ReferenceCompare.
